# Group 2 - Assignment 6: Disease Spread Simulation

## How to Run
1. Open `group_2_assignment6/group_2_assignment6.sln` in an IDE.
2. Make sure .NET 9.0 and MonoGame 3.8 are installed.
3. Click Run/Start to build and launch the simulation.

## Controls
- Click the **Replay** button on the right panel to restart the simulation.
- Press **Escape** to quit.
